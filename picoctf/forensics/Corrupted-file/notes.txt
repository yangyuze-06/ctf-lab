既然hints给了JPEG,我们不妨去找找那个JPEG对应的头部数字？

我们用xxd修复一下

1.导出hex
xxd file > file.hex

2.修改
sudo vim file.hex

3.hex的修改是要按下r,光标在你对应的那个想改的字上面，按下expected letter

4.退出后还原
xxd -r file.hex > fixed.jpg
