## 🧠 heapdump 文件分析方法

当 grep / strings 无法提取 flag 时：

### 方法：less 手动搜索

1. 打开文件

less heapdump.heapsnapshot


2. 搜索关键词

/picoCTF


3. 跳转
- n → 下一个
- N → 上一个

4. 常见搜索词
- picoCTF
- flag
- secret
- token

5. 退出

q


### ⚠️ 注意

- heapdump 是半文本文件
- grep 可能失效
- less 是最稳定方案
